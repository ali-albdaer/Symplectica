# Solar System Sim (Gen30)

No Node/NPM. Serve locally with Python:

- `python -m http.server 8000`
- Open `http://localhost:8000/`

Controls:
- Click: pointer lock
- `/`: Dev Menu (releases pointer)
- `T`: Telemetry overlay
- `WASD`: move
- `Space`: jump (walk mode) / up (free flight)
- `Shift`: down (free flight)
- `F`: toggle free flight
- `V`: toggle 1st/3rd person
- RMB: grab/release nearby object

If anything fails to load, open the on-screen Debug Log (top-right). Errors are also mirrored there.
